# 🚗 Car Rental 

Modern ve kullanıcı dostu bir araç kiralama platformudur. Kullanıcıların diledikleri araçları markasına, modeline ve yılına göre filtreleyebileceği, detaylarını inceleyebileceği interaktif ve responsive bir web uygulamasıdır.

## 🛠️ Kullanılan Teknolojiler

- **React 19**
- **TypeScript** 
- **Vite** 
- **Tailwind CSS**
- **React Router DOM** 
- **Axios** 
- **React Paginate** 
- **React Select** 
- **Motion** 

## ⭐ Özellikler

- 🚗 Geniş araç filosu listeleme
- 🔍 Marka, model ve yıla göre detaylı filtreleme
- 📄 Sayfalama (Pagination) sistemi ile kolay gezinme
- 📱 Responsive ve mobil uyumlu tasarım
- ⚡ Vite ile optimize edilmiş yüksek performans
- 🎨 Tailwind CSS ile modern ve şık arayüz
- 🔗 URL tabanlı filtreleme ve durum yönetimi
- ✨ Akıcı animasyonlar ve geçiş efekleri
- ⚠️ Hata yönetimi ve yükleme durumları
